## 心理学概论

### 心理学历史与流派
- 结构主义
- 功能主义
- 完形主义
- 精神分析
- 行为主义
- 人本主义

#### 结构主义
1. 心理学的对象为意识经验
2. 心理学的目的是揭示意识经验的结构
3. 用实验内省的方法去分析各种经验
4. 结构主义对于意识经验的分析结果是一切复杂的经验都可以被分析为感觉，简单的意象和简单的情感
5. 结构主义主张将科学与科学的应用区分开来

#### 功能主义（机能主义）
1. 心理学是关于意识或心理生活的科学
2. 心理学揭示有机体适应环境过程中，心理或意识的功能
3. 主张内省和实验，描述和解释心理生活的现象和条件
4. 意识流：意识是流动的、变化的
5. 实用主义，心理学揭示意识的功能是为了帮助个体有效地适应环境

#### 完形心理学
1. 格式塔心理学（）
2. 整体形态，整体组织
3. 反对元素分析，强调整体组织
4. 研究意识与行为
5. 不同的元素以不同的形式组合起来，它所代表的意义，远远大于这个元素单纯相加的和

#### 精神分析
1. 研究对象：潜意识
2. 方法：自由联想
3. 目的：了解观念和行为的潜意识根源
4. 心理结构观
    - 人的心理活动
        - 意识
            - 所思所想
            - 社会规范所接受
        - 潜意识
            - 本能冲动和欲望
            - 不被社会所接受
5. 防御机制
    - 退行防御机制
    - 合理化防御机制

> 防御机制是一种常见且重要的心理过程。个体在面临紧张或压力情境时，其内心会产生焦虑不安的情绪，这时个体会自觉或不自觉地采取一些策略来摆脱烦恼，减轻内心的不安，那他所采取的这些策略就是心理防御机制。


#### 行为主义
1. 研究对象：行为（外在的行为）
2. 行为由外部（环境）刺激引起
3. 刺激-反应（S-R公式）
4. 恐惧形成实验
5. 功：使用客观的研究方法，强化了心理学的自然科学特征
6. 过：将研究对象局限于行为，过于环境的作用，忽视遗传的因素


#### 人本主义
1. 心理学应该关注人的积极面
2. 促进人潜能的发挥和价值
3. 现象学方法
4. 需要层次理论模型


### 行为背后的动机

> 动机：能够引起、维持个体活动，并使活动朝着某一目标进行的内在动力。

- 始动功能
- 引导功能
- 维持和调节功能

> 需要：内部条件，是由于有机体内部缺乏某种东西而引起的一种心理的感受。

```
graph LR
A(缺乏)-->B(需要)
B-->C(紧张)
C-->D(动机)
D-->E(行为)
```

> 诱因：外部条件。是引起个体动机外部刺激或情境。

```
graph TD
A(诱因)-->B(正诱因)
A-->C(负诱因)
B-->D(趋近)
B-->E(接受)
C-->F(逃离)
C-->G(逃避)
D-->I(满足的刺激)
E-->I
F-->I
G-->I
```
